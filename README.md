**School Financial Sustainability Dashboard | Excel Portfolio Project
Excel-based Financial Sustainability Dashboard | Developed by Hafsa Mughal for a real-world school financial reporting workflow | Portfolio demonstration using realistic sample data**

**Problem Statement**
The school operates across 5 campuses, but there was no centralized financial reporting system to compare campus-wise financial performance.
Before this dashboard:
1.	No Cost vs Value analysis across campuses
2.	Revenue, salary, and expenses were maintained in separate departments
3.	No centralized financial summary for management
4.	Difficult to identify profitable and loss-making campuses
5.	Fee defaulter losses could not be monitored efficiently
   
_Note: This project uses realistic sample data for portfolio demonstration. Original organizational data has not been shared due to confidentiality._

**My Role**
As a Data Analyst, I:
1.	Identified business requirements with stakeholders
2.	Determined the required datasets for analysis
3.	Coordinated with Accounts and HR departments for data collection
4.	Designed a standardized reporting template
5.	Cleaned and transformed raw data
6.	Built automated summary tables
7.	Developed an interactive Excel dashboard with automatic updates

**Dataset**
The analysis was built using three operational datasets:
1.	Fee Collection – Student fee collection and defaulter information
2.	Salary – Employee salary, department, and active staff records
3.	Expenses – Monthly operational expenses by category
These datasets were integrated into a centralized Financial Summary table.

**Dashboard Features
KPI Cards**
1.	Total Fee Collected
2.	Total Cost
3.	Net Profit / Loss
4.	Active Staff
**Visualizations**
1.	Revenue vs Total Cost by Campus
2.	Net Profit / Loss by Campus
3.	Revenue Lost due to Fee Defaulters
4.	Expense Category Distribution
The Summary sheet is linked with all cleaned datasets, allowing the dashboard to update automatically whenever new monthly data is added.

**Key Insights (Sample Data)**
1.	F-8 Campus is operating at a financial loss.
2.	Marketing is the highest operational expense category.
3.	F-10 Campus has the highest revenue loss due to fee defaulters.
4.	Salary costs represent the largest share of total operating costs.
5.	The dashboard enables quick financial comparison across all campuses.

**Business Impact**
The dashboard provides management with a centralized reporting solution to:
1.	Compare financial performance across campuses
2.	Monitor revenue, expenses, and salary costs
3.	Identify financially underperforming campuses
4.	Track fee defaulter losses
5.	Support budgeting and financial planning
